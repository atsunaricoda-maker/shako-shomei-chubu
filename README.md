# 車庫証明かんたん作成（静岡県中部 ・ 静岡市・志太榛原 対応）

行政書士の方が、**スマホ1台で車庫証明（自動車保管場所証明）の書類一式**を作成できるツールの **静岡県中部版** です。
入力は1回だけ。申請書・自認書／使用承諾証明書・所在図／配置図・提出チェックリストへ自動転記します。

`index.html` の1ファイルだけで動きます。インストール不要・オフライン動作・**個人情報は端末内のみで処理**（外部送信なし）。

> 🌐 **公開URL（スマホで開く）: https://atsunaricoda-maker.github.io/shako-shomei-chubu/**
> このURLをスマホのブラウザで開き、「ホーム画面に追加」でアプリになります（PWA）。
>
> 西部版（磐田・浜松ほか）: https://atsunaricoda-maker.github.io/shako-shomei-iwata/

## 特長

- **1回入力 → 全書類に自動転記**（申請書 or 届出書／自認書 or 承諾書／所在図・配置図／チェックリスト）
- **普通車／軽自動車に対応**：種別を選ぶと様式・手数料（2,400円↔無料）・文面が自動で切替
- **軽の届出義務を地域で自動判定**：中部は**静岡市・藤枝市・焼津市が届出義務地域**（旧町村部に除外あり）。島田市・牧之原市などは届出不要、と自動表示
- **静岡県中部7警察署をプリセット**：提出先を選ぶだけで宛先・所在地・電話が全書類に反映
- **配置図をSVGで自動作図** / **本拠⇔保管場所の2km判定** / 車両がスペースに収まるか判定 / 和暦変換
- **案件の保存・読み込み**（端末内）/ **印刷・PDF保存**（A4・コンビニ印刷可）

## 静岡県共通の仕様（2026年6月時点・公式確認済み）

- 手数料 **2,400円**（保管場所証明手数料・静岡県**収入証紙**で納付）。**県内全署共通**（市が変わっても提出先の警察署が変わるだけ）
- **保管場所標章は2025年4月1日に廃止**（車庫シール・標章交付手数料なし）。軽自動車も届出のみ・手数料無料
- 本拠と保管場所は**直線2km以内**／押印は2021年〜不要

## スマホアプリとして使う（PWA）

HTTPSのURL（上記）をスマホで開き、ホーム画面に追加：
- **iPhone（Safari）**：共有 → 「ホーム画面に追加」
- **Android（Chrome）**：メニュー → 「アプリをインストール」

## 他の署を追加するには

`index.html` 冒頭の `STATIONS` 配列に1行追加（`kei` は軽の届出義務地域なら `"req"`、不要なら `"none"`）。
